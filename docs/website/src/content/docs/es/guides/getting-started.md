¡Bienvenido a la plantilla **Starter Gradle**! Esta guía te ayudará a configurar tu entorno de desarrollo y a poner en marcha el proyecto en minutos.

## Requisitos Previos

Antes de comenzar, asegúrate de tener instalado lo siguiente:
- **Java JDK 21** o superior.
- **Git**.
- Un shell compatible con bash (Linux, macOS o Git Bash en Windows).

## Instalación

1. Clona el repositorio:

   ```bash
   git clone https://github.com/dallay/starter-gradle.git
   cd starter-gradle
   ```

2. Ejecuta el comando de configuración:

   ```bash
   make setup
   ```

## Inicio Rápido

### Construir el proyecto

```bash
make build
```

### Ejecutar la aplicación

```bash
make run
```

### Ejecutar pruebas

```bash
make test
```
