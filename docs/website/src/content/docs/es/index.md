## ¿Por qué Starter Gradle?

<CardGrid stagger>
  <Card title="Lógica de Construcción Centralizada" icon="setting">
    Gestiona todos tus plugins de convención en un solo lugar, evitando la duplicación entre módulos.
  </Card>
  <Card title="Soporte Multilenguaje" icon="open-book">
    Desarrolla sin problemas con Kotlin, Java y Spring Boot utilizando módulos preconfigurados.
  </Card>
  <Card title="Altos Estándares de Calidad" icon="approve-check">
    Análisis estático integrado (Detekt, SpotBugs, PMD) y formateo automático (Spotless).
  </Card>
  <Card title="Herramientas Modernas" icon="rocket">
    Utiliza Catálogos de Versiones de Gradle, Caché de Configuración y dependencias actualizadas.
  </Card>
</CardGrid>
