## ¿Por qué Starter Gradle?

<CardGrid stagger>
  <Card title="Lógica de Build Centralizada" icon="setting">
    Gestiona todos tus plugins de convención en un solo lugar con `build-logic`. Sin copiar y pegar configuraciones entre módulos.
  </Card>
  <Card title="Soporte Multilenguaje" icon="open-book">
    Soporte de primera clase para **Kotlin**, **Java** y **Spring Boot** con módulos preconfigurados y plugins de convención.
  </Card>
  <Card title="Altos Estándares de Calidad" icon="approve-check">
    Análisis estático integrado con **Detekt**, **SpotBugs** y **PMD**. Formateo automático con **Spotless** para mantener tu código consistente.
  </Card>
  <Card title="Herramientas Modernas" icon="rocket">
    Utiliza **Catálogos de Versiones de Gradle**, **Caché de Configuración** y mantiene las dependencias actualizadas con buenas prácticas.
  </Card>
</CardGrid>
