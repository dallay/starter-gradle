## Why Starter Gradle?

<CardGrid stagger>
  <Card title="Centralized Build Logic" icon="setting">
    Manage all your convention plugins in one place, avoiding duplication across modules.
  </Card>
  <Card title="Multi-language Support" icon="open-book">
    Seamlessly develop with Kotlin, Java, and Spring Boot using pre-configured modules.
  </Card>
  <Card title="High Quality Standards" icon="approve-check">
    Integrated static analysis (Detekt, SpotBugs, PMD) and automated formatting (Spotless).
  </Card>
  <Card title="Modern Tooling" icon="rocket">
    Uses Gradle Version Catalogs, Configuration Cache, and up-to-date dependencies.
  </Card>
</CardGrid>
